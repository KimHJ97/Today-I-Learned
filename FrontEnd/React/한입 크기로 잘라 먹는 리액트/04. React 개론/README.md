# React 개론

Meta에서 개발한 오픈소스 Javascript 라이브러리로 대규모 웹 서비스의 UI를 더 편하게 개발하기 위해 만들어진 기술  

 - 컴포넌트를 기반으로 UI를 표현한다.
 - 화면 업데이트 구현이 쉽다.
 - 화면 업데이트가 빠르게 처리된다.

## 첫 React App 생성하기

 - 프로젝트 생성
```bash
npm create vite@latest

create-vite@5.5.2
Ok to proceed? (y) y
√ Project name: ... 푸로젝트명
√ Select a framework: » React
√ Select a variant: » JavaScript

cd 프로젝트
npm install
```
